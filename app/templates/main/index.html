This is the official website of the Love Living at Home Organisation. It employs a range of functionalities that makes the management of new and current members, volunteers and resources more efficient. Some of these functionalities include:

Administrator Dashboard

  • Account Management

    • Registered User - This displays a table where the user can view and manage currently registered users and other administrators.
    • Add New User - Provides a form that allows the user to create a new user account.
    • Invite New User - Allows the user to enter a new user name and email address and to send a request to a new user to create an account.

  • System Configuration

    • Edit Services - Allows the user to remove or edit a current service and add a new service.
    • Edit Service Categories - Allows the user to remove or edit a current service category or add a new service category.
    • Edit Metro Areas - Allows the user to edit or remove a current metro area or add a new metro area.
    • Edit Destination Address- Allows the user to view and manage the destination addresses.

  • Additional Actions

    • Generate Report - Allows the user to create a report of all the completed volunteer services by category.

Request Manager

  • Create Request

    • Provides a form to allow the user to create a request. The form can be of three types:

    • Transportation Request Form
    • Member's Home Request Form
    • Office Time Request Form

  • Search Request

    • Uses various filters to help the user search for a specific type of request

People Manager

  • Members

    • Add Member - Allows the user to add a new member to the Love Living at Home Members database.
    • Search Results - Displays a table showing all the members associated with Love Living at Home that suit a specific search criteria. Each member is portrayed as a card that allows the user to edit member information.

  • Volunteers

    • Add Volunteers - Allows the user to add a new volunteer to the Love Living at Home Volunteers database.
    • Search Results - Displays a table showing all the volunteers associated with Love Living at Home that suit a specific search criteria. Each volunteer is portrayed as a card that allows the user to edit volunteer information.

  • Local Resources

    • Add Local Resource - Allows the user to add a new affiliated organisation to the Love Living at Home Local Resource database.
    • Search Results - Displays a table showing all the organisations associated with Love Living at Home Each resource is portrayed as a card that allows the user to edit resource information.

Your Account

Allows the user to manage their account settings and change their login information.
